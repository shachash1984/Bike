If you want to use Realistic Engine Sounds with Edy's Vehicle Physics,
import Edy's Vehicle Physics, Realistic Engine Sounds and later EdysVP_RES-Lite.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Edy's Vehicle Physics. This prefabs are ready to use with Edy's Vehicle Physics 5.
If you don't own Edy's Vehicle Physics, you can buy it here: https://assetstore.unity.com/packages/tools/edy-s-vehicle-physics-403

Read the documentation for tutorials.
