If you want to use Realistic Engine Sounds with BoneCracker Games' Realistic Car Controller V3,
import Realistic Car Controller V3, Realistic Engine Sounds and later RCCV3_RES-Lite.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Realistic Car Controller V3. This prefabs are ready to use with Realistic Car Controller V3.
If you don't own Realistic Car Controller V3, you can buy it here: https://www.assetstore.unity3d.com/en/#!/content/16296

Read the documentation for tutorials.
